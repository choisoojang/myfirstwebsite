깃 허브(GitHub Pages) 활용한 배포 가이드

GitHub로 나만의 웹사이트를 배포해보세요!

GitHub는 우리가 만든 웹 페이지를 웹 서버에 자동으로 올려주는 시스템을 제공합니다.

1. 준비물

  • GitHub 계정
  • HTML, CSS, JS 파일이 있는 저장소

1-1. 회원가입하기

  • https://github.com/
  • 위의 링크에 들어가서 "Sign in" 클릭

1-2. 저장소에 웹 사이트 파일 넣기

  • Top repositories 옆에 있는 New 버튼 클릭
  • 저장소 이름을 입력하고 저장소 만들기
  • 저장소에 우리가 만든 웹 페이지 업로드하기
  • 단, 웹 페이지 파일 이름은 index가 되도록 하기(index.html)

2. 설정 방법

  1. 저장소에 들어가서 Settings 클릭
  2. 좌측 메뉴에서 Pages 선택
  3. Branch는 main, /(root)로 설정
  4. Save 클릭 후 몇 초 기다리기

3. 나의 배포 주소 확인
